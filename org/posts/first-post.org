* Why Org Mode?

Because markdown is for corporate simps. Org mode gives you:

- Real structure and outlining
- Code blocks that actually work  
- Math support: $E = mc^2$
- Tables that don't suck
- Export to anything (HTML, LaTeX, PDF)
